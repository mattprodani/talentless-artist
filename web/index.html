Not A Gallery


My attempt to try to paint while having no painting skills at all.

The Works

Abstract

None of these images are real paintings. These images are fully computer generated by a model trained to paint like Van Gogh.
Press shuffle to discover all the paintings created by this model.
This project was made using StyleGAN2 trained through real artwork.
